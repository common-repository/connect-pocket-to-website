== Description ==

This plugin allows you to display your Pocket list into your WordPress site

= Need Help? Found A Bug? Want To Contribute Code? =

Support for this plugin is provided via the [WordPress.org forums](https://wordpress.org/support/plugin/connect-pocket-to-website).

The source code for this plugin is available on [GitHub](https://github.com/JeanDavidDaviet/connect-pocket-to-website).

== Installation ==

1. Go to your admin area and select Plugins â†’ Add New from the menu.
2. Search for "Connect Pocket To Website".
3. Click install.
4. Click activate.
